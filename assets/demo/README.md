# Demo Assets

This folder contains demo assets for GitHub repository presentation.

## Files

- `demo-preview.gif` - Main demo animation for README
- `screenshot-main.png` - Application main interface
- `screenshot-alert.png` - Alert system in action
- `logo-banner.png` - Repository banner image

## Usage

These assets are used in:
- README.md header section
- GitHub repository social preview
- Documentation illustrations
- Release announcement materials

## Creating Demo Assets

To create your own demo assets:

1. **Screenshot the application** running with different alert states
2. **Create GIF animation** showing the detection in action
3. **Design banner image** with project logo and key features
4. **Optimize file sizes** for web display

## File Requirements

- **GIF**: Max 10MB, optimize for web
- **PNG**: Use for screenshots, compress for GitHub
- **Banner**: 1200x630px for social media preview
